You are helping me build a personal portfolio for Ganesh Singha, a Full Stack Developer.

─── CONCEPT ─────────────────────────────────
The portfolio is themed as the Antigravity IDE — a fictional IDE that looks and feels
exactly like VS Code's default dark theme, but named "Antigravity". The critical design
rule is: each portfolio section (Home, About, Projects, Skills, Contact) is a
SEPARATE open file, not a scroll section on one page. Clicking a file in the sidebar
NAVIGATES to a new route/page — it does NOT smooth-scroll. Each page renders its own
content inside the "editor area", exactly like switching tabs in VS Code.

This means the portfolio is a multi-page Next.js app where each page looks like
a different file is open in the same IDE window.

─── TECH STACK ──────────────────────────────
Framework  : Next.js 14, App Router, TypeScript
Styling    : Tailwind CSS v3
Fonts      : next/font — "JetBrains Mono" (all text, matching VS Code feel)
Icons      : lucide-react (IDE chrome only)
Routing    : Next.js file-based routing — one route per "file"
               app/page.tsx          → home.tsx tab
               app/about/page.tsx    → about.html tab
               app/projects/page.tsx → projects.js tab
               app/skills/page.tsx   → skills.json tab
               app/contact/page.tsx  → contact.css tab
